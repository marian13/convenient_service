Overview

Converts strings to UpperCamelCase (capitalize_first_letter option is set to true by default). Implementation is very basic just to serve the need of this library. More comprehensive solution can be found in Rails, for example:

def call
  camelized = string.split(/[:_?!\-\ ]/).map { |part| upcase_first_char(part) }.join

  camelized[0] = camelized[0].downcase unless capitalize_first_letter

  camelized
end
